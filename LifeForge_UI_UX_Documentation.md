# LifeForge -- UI/UX Documentation (Dark Theme)

## 1. UI Vision & Design Principles

LifeForge uses a clean, professional dark theme focused on clarity,
motivation, and calm control.\
\
Design principles:\
- Minimal, distraction-free layouts\
- Dark-first UI to reduce eye strain\
- Soft gradients and subtle glow accents\
- Clear hierarchy and readable typography\
- Motion used sparingly and meaningfully

## 2. Color System

Primary Background:\
- Deep charcoal / near-black (#0E0E10)\
\
Secondary Surfaces:\
- Dark gray cards (#16181D)\
\
Primary Accent:\
- Energy Orange / Ember (#FF6A00)\
\
Secondary Accent:\
- Soft Red for alerts (#FF4D4D)\
\
Text Colors:\
- Primary: Off-white (#EDEDED)\
- Secondary: Muted gray (#9AA0A6)

## 3. Typography

Primary Font:\
- Inter or SF Pro (Android-compatible)\
\
Font Weights:\
- Headings: SemiBold / Bold\
- Body: Regular / Medium\
\
Text Rules:\
- Avoid all caps except labels\
- Line height 1.4--1.6

## 4. Core Screens Overview

Screens included in MVP:\
- Splash & Onboarding\
- Authentication (Login / Signup)\
- Dashboard (Coins, Progress, Status)\
- Exercise (Camera -- Push-up Detection)\
- Blocked Apps Manager\
- Lock Overlay Screen\
- Settings & Permissions

## 5. Onboarding Screens

Purpose:\
- Explain concept simply\
- Set trust for permissions\
\
Screens:\
1. Earn access through movement\
2. Block distracting apps automatically\
3. Your data stays on your device

## 6. Authentication Screen

Style:\
- Dark gradient background\
- Centered card layout\
\
Elements:\
- Email / Password\
- Continue with Google\
- Minimal text and soft glow buttons

## 7. Dashboard Screen (Home)

Main Elements:\
- Coin balance (large, central)\
- Daily progress bar\
- Step count summary\
- Status message (e.g., \'You\'re on fire\')\
\
Layout:\
- Card-based vertical stack\
- Rounded corners (16--20px)

## 8. Exercise Camera Screen

Layout:\
- Full-screen camera view\
- Top: Rep counter\
- Bottom: Start / Stop button\
\
Visuals:\
- Minimal overlays\
- Rep count animation on success

## 9. Blocked Apps Manager

Layout:\
- List of installed apps\
- Toggle to block/unblock\
\
Design:\
- App icons with muted grayscale\
- Blocked apps highlighted with accent color

## 10. Lock Overlay Screen

Purpose:\
- Interrupt access calmly but firmly\
\
Elements:\
- Message: \'Earn access to continue\'\
- Coin balance display\
- Button: Start Exercise\
- Button: Emergency Unlock\
\
Design:\
- Full-screen dark blur\
- Strong accent button

## 11. Settings & Permissions

Sections:\
- Permissions status\
- Privacy & data usage\
- App preferences\
\
Design:\
- Simple list layout\
- Clear permission indicators

## 12. Motion & Feedback

Animations:\
- Subtle progress fills\
- Soft scale on rep count\
\
Haptics:\
- Light vibration on rep completion\
- Strong vibration on unlock

## 13. Accessibility

\- Minimum contrast ratios\
- Large tap targets\
- Clear error messages

## 14. UI Constraints (Important)

\- No visual clutter\
- No ads in core UI\
- No popups except lock overlay\
- One primary action per screen
